## Goutside

### Deskripsi Website

Sistem persewaan gear outdoor yang bertujuan untuk memudahkan pengguna dalam melakukan persewaan alat outdoor dengan online. Dirancang agar pengguna dapat melakukan checking katalog, booking, dan pembayaran. Selain itu sistem ini juga dapat diakses oleh admin yang dapat melakukan

---

## Tech Stack yang Digunakan

- **Frontend** : HTML, CSS, Tailwind CSS, JavaScript, Vite
- **Backend** : Laravel
- **Database** : MySQL
- **Version Control** : Git & GitHub

---

## Cara Menjalankan Website

### Clone Repository
```bash
git clone https://github.com/Btenk27/Goutside.git
```
### Setelah Clone
```bash
cd Goutside/laravel
```
### buka di VSCODE
```bash
composer install
npm install
cp .env.example .env
```
### lalu sesuaikan pada .env
```bash
DB_DATABASE=goutside
DB_USERNAME=root
DB_PASSWORD=
```
### generate KEY
```bash
php artisan key:generate
```
### migrasi database
```bash
php artisan migrate
```
### jalankan server
```bash
php artisan serve
npm run dev
###jalankan di http://127.0.0.1:8000
```
